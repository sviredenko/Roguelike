Игра реализует:
Всю базовую часть .+источники света(изменение света соседних тайлов)+плавное движение камеры за игроком + повороты игрока всторну движения+вывод верхней панельки (в будущем будем выводить там жизни и инвентарь).

Чтение карты:
    @-игрок 
    x- дверь
    .-пол травой
    s- пол песком
    t-камень через него не можем пройти
    #- стена через нее тоже не можем пройти
    w- вода умираем
    1,2 - число которое мы можем пройти по тайтлу после чего он исчезает и если наступим на тайтл умрем
    f- огонь(источник света)
